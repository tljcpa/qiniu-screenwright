## 这是什么

Screenwright 不是"切块丢给大模型"的转换器，而是一个**编剧工作台**：

- **结构化剧本（YAML）**：人物、场景、动作、对白、转场，全部结构化，可机读可手改。
- **跨章人物一致性**：先建"故事圣经"（人物/地点/时间线），生成每一场都以它为单一事实源，杜绝前后矛盾。
- **行级双向溯源**：剧本每一行都能点回它来自的小说原文段落，作者可审计、可打磨。
- **可控目标媒介**：同一部小说，一键切换电影 / 剧集 / **短剧**，输出不同节奏的剧本。
- **内心戏外化**：自动识别不可拍的心理描写，转成动作 / 潜台词，并显式标记。
