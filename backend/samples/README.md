# 演示素材说明（版权与用途）

本目录是 Screenwright 的演示输入素材。两份，各司其职。

## 1. `english_pride_and_prejudice_ch1-3.txt`

- 《傲慢与偏见》(Pride and Prejudice)，简·奥斯汀，1813 年出版。
- **公有领域**（作者逝世逾百年）。原文逐字取自 Project Gutenberg 电子书 #1342（https://www.gutenberg.org/ebooks/1342）。
- 用途：**可信度主素材**。证明工具能处理真实、知名、未经我方改写的文学作品，而非过拟合到自造输入。对白密集、人物众多，适合演示跨章人物追踪与场景切分。

## 2. `中文网文样本_旧城咖啡.txt`

- **原创作品**，作者 tljcpa，专为本项目撰写并授权作为测试输入。人物情节均属虚构。
- **零版权风险**（自有原创内容，非提交的产品代码，不涉及代码重复率）。
- 用途：**短剧改编金句素材**。刻意设计了：
  - 两位主角（林深 / 沈言）跨三章身份一致 —— 演示故事圣经与一致性。
  - "铜铃 / 那封信 / 雨夜窗口" 跨章回扣线索 —— 演示长程关联。
  - 大段心理描写（第一章林深、第二章沈言）—— 演示"内心戏外化"。
  - 时间线"三年前雨夜 → 三天前 → 次日傍晚" —— 演示连贯性检查。
  - 情感钩子与反转 —— 适配中文短剧市场的改编演示（创新点①）。
